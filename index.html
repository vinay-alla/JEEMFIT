🚀 Get Started

UP AND RUNNING IN 60 SECONDS

Clone, install, configure, and launch. That's it.

bash — terminal
$git clone https://github.com/your-username/jeemfit.git
$cd jeemfit
$npm install # install all dependencies
$cp .env.example .env # configure your environment
GOOGLE_CLIENT_ID=your_id
DATABASE_URL=postgresql://...
JWT_SECRET=your_secret
$npm run dev # start development server
Prerequisites: Node.js 18+, PostgreSQL or MongoDB, a Google Cloud project with OAuth 2.0 credentials configured. See docs/setup.md for the full environment variable reference and database migration guide.
